This project is intend to check ADC Peripheral of ARM7 LPC2148,
A POT is connected to ADC0 channel3.

This program reads analog data over channel 3 of ADC0, \
converts it into digital value and display same decimal equivalent onto a LCD. 

Find adc.bin (binary) and adc.hex (hex file) in the Obj folder
